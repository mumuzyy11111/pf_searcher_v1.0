- `web/assets/js/conditions.js` 与 `web/assets/css/conditions.css`：状态查询交互与样式。
- `web/spells_filter.html`：组合式法术筛选页面。
- `web/assets/js/spell-filter-core.js`、`web/assets/js/spell-filter.js`：组合筛选逻辑。
- `tools/repair_content_integrity.py`：内容完整性修复工具，用于法术职业别名归一、法术职业/环位去重和专长文本污染修复。

### 使用说明

默认只生成审计报告：

```powershell
.\Apply-Repair.ps1
```

确认审计结果后再写入修复：

```powershell
.\Apply-Repair.ps1 -Apply
```
